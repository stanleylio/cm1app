Technology


System Overview

The custom data loggers are installed at each ocean-facing makaha, as shown in the satellite photo of the fishpond. In addition to logging the collected data locally, readings are transmitted to an x86 base station nearby via 900MHz XBee transceivers. The base station uploads the received data periodically to a server at the University of Hawaii. Measurements are then processed and made publicly available.

Implementation

Hardware

Research-grade oxygen optodes as well as a multitude of low-cost commercial-off-the-shelf sensors are employed. Beaglebone Black single-board computers aided by Arduinos control and query the sensors for readings according to a set schedule.

Software

sample scheduling; retry on failure; gracefully handle sensor failure; communication (error detection, "ping")

IO/control: Python; storage: SQLite; analysis and visualization: matplotlib, Plotly, and MATLAB; presentation: bootstrap + jQuery

Power

All nodes are powered by solar panels with battery backup. Depending on power consumption, battery size ranges from portable USB packs to lead acid car batteries are used.

Low cost alternatives

anti-biofouling + waterproofing

Bench Tests

characterize and compare low-cost MEMS sensors: accuracy, resolution/precision, noise/variance, drift/bias/stability, ease of deployment (mechanical: waterproofing)
