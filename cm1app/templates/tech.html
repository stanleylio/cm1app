Technology


System Overview

The custom data loggers are installed at each ocean-facing makaha, as shown in the satellite photo of the fishpond. In addition to logging the collected data locally, readings are transmitted to an x86 base station nearby via 900MHz XBee transceivers. The base station uploads the received data periodically to a server at the University of Hawaii. Measurements are then processed and made publicly available.

Implementation

Hardware

Research-grade oxygen optodes as well as a multitude of low-cost commercial-off-the-shelf sensors are employed. Beaglebone Black single-board computers (SBCs) aided by Arduinos control and query the sensors for readings according to a set schedule.

Software

The primary tasks for the Python programs on the SBCs are:

  • Regularly read the sensors, store and broadcast the collected data
  • Gracefully handle malfunctioning / decommissioned sensor
  • Respond to base station query for unscheduled sampling

Collected data are stored locally on each node as well as transmitted to nearby site base station. Base stations store and relay collected data to a UH web server for processing and presentation.

Software and programming languages used:

  • I/O control: Python, C
  • Data storage: SQLite
  • Analysis and visualization: matplotlib, plotly and MATLAB
  • Web presentation: bootstrap + jQuery/AJAX + Flask (Python)

Power

All nodes are powered by solar panels with battery backup. Depending on power consumption, batteries ranging from portable USB battery pack to sealed lead acid car battery are used.
