About this page

This page shows the system status in real-time. Click on any row to see recent readings and plots from that node. For information about this project, click here.


How to read the table

Each row in the table corresponds to a sensor package (called a "node"). A row turns red if there is no recent reading from that node (e.g. the node ran out of power). The row is yellow if any reading from that node is out of range (e.g. a bad sensor). The row is green otherwise.
