About this page

The table shows the current status of the SMART Coastlines Monitoring System. Click on any row to see recent readings and plots from that node. To learn more about this project, click here.

Each row in the table corresponds to a sensor package (called a "node"). A row turns red if there is no recent reading from that node (e.g. the node is offline for maintenance). The row is yellow if any reading from that node is out of range (e.g. biofouling or in need of calibration). The row is green otherwise.
